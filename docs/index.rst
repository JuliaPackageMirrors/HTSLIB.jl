Welcome to HTSLIB.jl's documentation!
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

HTSLIB.jl is a julia wrapper of `htslib https://github.com/samtools/htslib`.
Current support features include:

- bam read and write
- sam read and write

features under development include:

- bam query

For more detatails, see documentation below.
